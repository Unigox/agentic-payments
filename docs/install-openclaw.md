# Installing on OpenClaw

## How It Works

OpenClaw discovers skills via a `SKILL.md` file at the repository root. This repo already includes the required `SKILL.md`.

## Setup

1. Point your OpenClaw agent at this repository
2. OpenClaw automatically detects and loads the skill
3. No additional install steps required

## Verify

Tell your agent:

```
send money using Agentic Payments
```

The sign-in setup flow should begin.
